# Dekret
Dekret adalah sebuah *game* strategi berbasis-*CLI* yang dikembangkan menggunakan bahasa pemrograman C++. Dalam *game* ini, pemain memegang kuasa atas Indonesia yang ada di ujung tanduk pada tahun 2039. Tujuan pemain adalah untuk memimpin negeri ini selama satu periode, yakni hingga tahun 2044.
